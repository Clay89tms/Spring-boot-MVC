Необходимо создать два приложения (на разных портах)
Одно и второе должны иметь схожие public API
Цель - протестировать взаимодействие между этими сервисами используя один из инструментов
	- RestTemplate
	- FeignClient
Одно из приложений должно выполнять стандартные CRUD операции (получение списка всех объектов,
    получение одного по ид, сохранение, обновление и удаление). Для этого необходимо подключить БД (можно взять H2)

Второе приложение должно использовав один из выше перечисленных способов, соединиться с первым и через рест
    имплементировать все ендпоинты (аналог адаптера).

Покрыть основное приложение (фасад) swagger аннотациями. Убедиться что swagger страница отображает корректные данные.
